# Rule 13: Direction

The **-n** ending in Esperanto is used to indicate movement toward a destination.

### Examples:
- **La reĝino iris Londonon.** → The queen went to London.
- **La kato saltis sur la tablon.** → The cat jumped onto the table.
- **Mi ĵetis la pilkon en la akvon.** → I threw the ball into the water.

If the sentence already uses a preposition showing motion, the **-n** ending is not needed:
- **Mi sendis la leteron al ŝi.** → I sent the letter to her.
