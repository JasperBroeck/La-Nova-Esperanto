# Rule 4: Numerals

Esperanto numerals are straightforward and do not change form. The basic numbers are:

**unu, du, tri, kvar, kvin, ses, sep, ok, naŭ, dek, cent, mil**

## Special Forms:
- **Ordinal numbers**: Add "-a" → **kvara** (fourth)
- **Multiples**: Add "-obl" → **duobla** (double)
- **Fractions**: Add "-on" → **duono** (half)
- **Collectives**: Add "-op" → **duopo** (a pair)
- **Distribution**: Use "po" → **po tri pomoj** (three apples each)

## Example Sentences:
- **Mil naŭcent naŭdek kvin** → 1995
- **La kvina trono** → The fifth throne
- **Mi donis al ili po tri pomojn.** → I gave them three apples each.
