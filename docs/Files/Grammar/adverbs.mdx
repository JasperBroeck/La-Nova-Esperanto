# Rule 7: Adverbs

Adverbs in Esperanto always end in **-e**. Their comparative and superlative forms follow the same rules as adjectives.

## Comparison Forms:
- **Comparative:** Use "pli" (more) → **pli rapide** (faster)
- **Superlative:** Use "plej" (most) → **plej rapide** (fastest)

## Example Sentences:
- **La kato rapide kuris.** → The cat ran fast.
- **La hundo pli rapide kuris ol la kato.** → The dog ran faster than the cat.
- **Sed la gepardo plej rapide kuris el ĉiuj.** → But the cheetah ran fastest of all.
