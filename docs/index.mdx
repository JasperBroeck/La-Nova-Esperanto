# Esperanto Cheatsheet
## Grammatical endings, affixes and correlatives
"-a" -> adjective
"-e" -> adverb
"-j" -> plural
"-n" -> accusative
"-o" -> noun
"-i" -> infinitive
"-as" -> present verb
"-is" -> past verb
"-os" -> future verb
"-us" -> conditional
"-u" -> imperative

### Participles
|         | past  | present | future |
| ------- | ----- | ------- | ------ |
| active  | -int- | -ant-   | -ont-  |
| passive | -it-  | -at-    | -ot    |

### Fixes
Prefixes:
- "bo-" -> relation by marriage, in-law
- "ĉef-" -> head, chief
- "dis-" -> separation, scattering
- "ek-" -> sudden or momentary action
- "ge-" -> both sexes together
- "mal-" -> opposite
- "re-" -> over again, back again
Suffixes:
- "-adi" -> to do an action persistently or repeadetedly
- "-ado" -> frequent or continuous action
- "-aĵo" -> thing, substance
- "-ano" -> member, inhabitant
- "-aro" -> collection, group
- "-ebla" -> possible
- "-eco" -> abstract quality
- "-eg-" -> great size, intense degree, augmentative
- "-ejo" -> place
- "-ema" -> propensity, tendency
- "-estro" -> leader, chief
- "-et-" -> smallness, small degree, diminutive
- "-ido" -> child, descendent
- "-igi" -> to make, to cause to become
- "-iĝi" -> to become
- "-ilo" -> tool, instrument
- "-inda" -> worthy of
- "-ino" -> feminine
- "-ismo" -> doctorine
- "-isto" -> person connected with something (instruisto, komunisto)
- "-obla" -> multiple (duobla, triobla, kvarobla)
- "-ona" -> fraction (duona: half, trione: a third)
- "-ujo" -> container, country (countries are more often used with "-io")
- "-ulo" -> person prossessing a quality or characteristic (riĉulo: rich person, junulo: young person)
- "-um-" -> indefinite suffix indicatting some relation with the root (malvarmumo: a cold, kolumo: collar)
